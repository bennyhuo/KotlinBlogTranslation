### 注意事项

* 翻译时直接修改对应译文，不要修改格式。
* 如需在发布后接受打赏，可以将 reward 设置为 true，并且提供微信和支付宝二维码图片的地址（可以外链，也可以直接上传到本仓库 /assets/文章名/ 目录，地址写绝对路径，比如 /assets/文章名/wechat.png）。
* 4.27 新增译者展示，将你的名字和主页地址写入文章开头的属性，博客中会直接展示，下面是个例子：

